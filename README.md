# Projet WordPress — Module C741

Site de présentation du projet WordPress pour le module C741 (Projet interdisciplinaire — CMS) à l'ESIG.

## Contenu

- **Concept** : Créer un site WordPress professionnel sur un sujet au choix
- **Objectifs** : Architecture, charte graphique, contenus, responsive, performance, UX
- **À rendre** : Site en ligne sur Infomaniak + présentation en classe
- **Exigences** : Thème personnalisé, contenus variés, responsive, cache, Lighthouse
- **Planification** : 6 séances avec jalons + devoir vacances
- **Évaluation** : 20 pts projet + présentation
- **Support & IA** : Règles d'utilisation de l'IA
- **Checklist** : Auto-évaluation avant rendu

## Déploiement

Le site est déployé automatiquement via GitHub Pages.

**URL** : https://fallinov.github.io/esig-sfa-741-projet/
